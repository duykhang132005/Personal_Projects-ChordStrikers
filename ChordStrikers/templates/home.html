Project Info

Inspiration

This project is a homemade application meant to assist in transcribing music and also chord-reading for quick self-accompaniment. The inspiration comes from a wide range of existing software like MuseScore and Chordify that highlights what chords to play, but aims to improve such an experience with more interactive features without a pay wall.

Technologies Used

Will add as we go, but generally, basic front-end and backend like JS, HTML and CSS. Will look into more UI/UX designs for mobile and other interfaces soon. To be updated.

Future Plans

Keep making this more accessible. This starts out as a personal tool, but I hope it may transform into something beneficial for more people as time goes on.
